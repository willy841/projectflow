# Event Agent Skills Plan

## Scope
Create a first-pass skill architecture for a physical event planning agent focused on year-end parties, spring banquets, corporate dinners, award ceremonies, and internal celebration events.

## Proposed Skills
1. event-brief-clarifier
2. event-theme-designer
3. event-flow-architect
4. event-ops-planner
5. event-budget-planner
6. event-proposal-writer

## Deliverables
- A skill map with each skill's purpose, inputs, outputs, and boundaries.
- Draft `SKILL.md` content for each skill so they can later be installed into a real skills directory.
- A suggested parent agent role definition that uses these skills.
