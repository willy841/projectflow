# 欄位級連動地圖（總覽）

這一版不是講「哪個頁面接哪個頁面」，而是講：

> **你改了哪個欄位，哪些地方會一起變。**

## 閱讀方式

每張卡只看 4 件事：
1. 你在哪裡改
2. 寫到哪個 table.column
3. 哪些頁面 / 區塊會變
4. 什麼條件下才成立
