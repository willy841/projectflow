# Projectflow 連動地圖

這不是規格書，而是給你快速看懂 **頁面怎麼互相影響** 的入口。

## 先看這裡

- [主流程總圖](./projectflow-overview.md)
- [設計線](./design-line.md)
- [備品線](./procurement-line.md)
- [廠商線](./vendor-line.md)
- [財務 / 報價 / 結案線](./financial-line.md)

## 怎麼讀

每一張圖都只看 4 件事：
1. 從哪個頁面開始
2. 按了哪個動作才會成立
3. 資料會承接到哪裡
4. 最後你在哪個頁面看得到結果

## 這份圖的目的

不是要取代 code，而是讓你：
- 一眼看懂上游 / 下游
- 知道某個數字是從哪裡來
- 知道按下「全部確認」後，哪些頁面會被影響
