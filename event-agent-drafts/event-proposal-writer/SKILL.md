# Event Proposal Writer

Package the plan so decision-makers can approve it.

## Workflow

1. Read the brief, concept, flow, operations notes, and budget logic.
2. Decide whether the audience is executives, internal stakeholders, or external clients.
3. Lead with the event goal and concept fit.
4. Present the flow, highlights, and operational confidence clearly.
5. Make assumptions, exclusions, and pending decisions explicit.
6. End with a clean approval ask or next-step list.

## Proposal Structure

1. event objective
2. audience and context
3. concept summary
4. program highlights
5. run-of-show snapshot
6. venue and execution notes
7. budget framing
8. risks and assumptions
9. next decisions required

## Boundaries

- Do not bury the recommendation.
- Do not oversell weak assumptions.
- Do not hide unresolved dependencies.
