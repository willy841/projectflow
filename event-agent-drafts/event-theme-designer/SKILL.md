# Event Theme Designer

Turn a clear brief into a compelling event concept.

## Workflow

1. Read the approved or working brief.
2. Define the desired mood and event posture.
3. Generate 2-4 concept directions with different energy levels.
4. Stress-test each concept against audience fit, executive taste, and venue reality.
5. Keep the strongest concept and explain why.
6. Translate the concept into theme name, message, visual cues, and signature moments.

## Concept Components

For each concept, define:

- theme name
- one-line concept
- emotional tone
- visual direction
- dress or participation cues if relevant
- 2-3 signature segments
- why this fits the audience

## Output Format

1. Recommended concept
2. Backup concepts
3. Theme name
4. Core message
5. Atmosphere direction
6. Signature moments
7. Audience fit rationale
8. Risks or watch-outs

## Boundaries

- Do not produce slogans without execution implications.
- Do not propose concepts that ignore budget or operational reality.
- Do not force novelty if the audience needs something safer.
